# zero-oracle

Decentralized AI prediction oracle built on 0G infrastructure. Uses 0G Storage for permanent audit trails and 0G Compute Router for AI-powered prediction market analysis.

## Architecture

- **Smart Contracts** (0G Chain) — On-chain prediction registry and agent reputation tracking
- **AI Agent** (Node.js) — Analyzes news, social sentiment, and market data via 0G Compute Router
- **Storage Layer** (0G Storage) — Permanent, verifiable audit trail for every prediction
- **Frontend** (React) — Live dashboard, historical predictions, audit log viewer

## Stack

- 0G Chain (EVM-compatible L1, 11k TPS, sub-second finality)
- 0G Storage (decentralized, AI-optimized, 95% cheaper than AWS)
- 0G Compute Router (OpenAI-compatible decentralized inference)
- Solidity + Hardhat
- Node.js + Express
- React
